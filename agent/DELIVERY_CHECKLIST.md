## 🚀 快速启动

### 1. 编译项目
```bash
cd server
mvn clean install
```

### 2. 启动服务
```bash
mvn spring-boot:run
```

### 3. 运行测试
```powershell
.\test-pages.ps1
```

### 4. 访问页面
在浏览器中打开 http://localhost:8080/
